# Infinity Telecom — Theme + All Numbers

- ☀️ Light Theme / 🌙 Dark Theme
- Theme পছন্দ Browser-এ Save থাকবে
- 📋 All Numbers বাটন
- সব সেভ করা Customer Number দেখা
- Search
- শেষ Amount
- মোট ব্যবহারের সংখ্যা
- Number-এ ক্লিক করলে Number ও Amount অটো পূরণ
